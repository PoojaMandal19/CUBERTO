# 🚀 Cuberto Website Clone

A modern and responsive clone of the Cuberto landing page built using **HTML, CSS, JavaScript, GSAP, and Locomotive Scroll**. This project focuses on smooth scrolling, premium animations, interactive UI elements, and modern web design.

🔗 

## 📂 GitHub Repository

🔗 https://github.com/PoojaMandal19/Cuberto-Clone

## ✨ Features

- 🎨 Modern UI inspired by Cuberto
- 📱 Fully Responsive Design
- ⚡ Smooth Scrolling (Locomotive Scroll)
- 🎬 GSAP Scroll Animations
- 🖱️ Custom Cursor Animation
- 💫 Interactive Hover Effects
- 🎥 Video Sections
- 📜 ScrollTrigger Animations
- 🚀 Optimized Performance

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP
- ScrollTrigger
- Locomotive Scroll

## 🚀 Installation

Clone the repository

git clone https://github.com/PoojaMandal19/Cuberto-Clone.git

Open the project

cd Cuberto-Clone

Run the project

Simply open **index.html** in your browser.
